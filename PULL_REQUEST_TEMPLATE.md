# Add Teal & Charcoal color system

This PR adds a Teal & Charcoal color system to the repository and a small set of UI tokens and example styles.

Files added:
- assets/colors.css — CSS variables for light and dark themes, plus small UI tokens (buttons, cards, badges)

Usage
- Link assets/colors.css from your HTML or include it in your build pipeline.
- Toggle dark mode by adding data-theme="dark" to the <html> or <body> element.

This is a small, non-breaking change that only adds a new asset and does not modify existing files.
